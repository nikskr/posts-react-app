# 📝 Posts React App

**🔗 Live Demo:** [https://nikskr.github.io/posts-react-app/](https://nikskr.github.io/posts-react-app/)

A feature-rich Single Page Application (SPA) built with React, focusing on efficient data handling, custom UI components, and modern routing practices.

## 🚀 Key Features

### 🎨 UI & Styling
*   **Styling Architecture:** Implemented using **CSS Modules** for component-level scoping and standard CSS.
*   **Custom UI Library:** Developed a reusable component library (Buttons, Inputs, Modals) from scratch.
*   **Animations:** Smooth interface transitions powered by **React Transition Group**.
*   **Modals:** Accessible and reusable modal window system.

### ⚙️ Core Logic & State Management
*   **Component Patterns:** Managed both **Controlled and Uncontrolled** components.
*   **State Flow:** Efficient data flow using **Lifting State Up** and managing **Props Drilling**.
*   **Global State:** Implemented **Context API (useContext)** for application-wide state management.
*   **Optimization:** Enhanced performance using **Memoization** (useMemo) to prevent unnecessary re-renders.
*   **Custom Hooks:** Extracted complex logic into **Custom Hooks** for cleaner decomposition.

### 🔍 Data Handling
*   **API Integration:** Dynamic data fetching from **JSON Placeholder**.
*   **Search & Filtering:** Real-time search functionality and multi-criteria filtering.
*   **Sorting:** Flexible **Dropdown Menus** for data organization.
*   **Pagination:** Traditional page-based navigation and **Infinite Scroll** for seamless browsing.

### 🚦 Routing & Security
*   **Navigation:** Robust routing system using **React Router**.
*   **Authentication:** Simulated user authorization flow.
*   **Access Control:** Clear separation between **Private and Public routes**.
